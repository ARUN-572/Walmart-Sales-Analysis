# Walmart Sales Analysis

## Project Objective

Analyze Walmart sales data to identify trends in sales,
profit, customer behavior, and regional performance.

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- SQL

## Dataset

Customer and sales transaction data.

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Dashboard Creation
5. Business Insights

## Key Insights

- West region had the highest sales.
- Technology generated the highest profit.
- December recorded peak sales.
